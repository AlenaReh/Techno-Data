## Description 

This is a CMS-style blog site, where developers can publish their technical knowledge and thoughts with each other.

* Visual Studio
* Node.js
* Javascipt
* Bulma

* Handlebars
* MySQL2
* MYSQL Workbench
* Sequelize
* API
* Insomnia
* Git Bash
* Heroku

## Usage

This application allows people with same interests share their thoughts and ideas.
